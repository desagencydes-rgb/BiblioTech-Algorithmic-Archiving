# BiblioTech: Algorithmic Archiving

A modern JavaFX Laboratory Management System designed for efficient library administration.

## Features

- **Dashboard**: Real-time overview of library statistics.
- **Member Management**: Add, edit, and track library members.
- **Book Management**: Catalog and organize books.
- **Loan System**: Track borrowing and returns (`Emprunts`).
- **AI Assistant**: Helper tool for library tasks.
- **Settings**: Configurable application preferences.

## Getting Started

### Prerequisites

- Java JDK 11 or higher
- JavaFX SDK (if not bundled)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/desagencydes-rgb/BiblioTech-Algorithmic-Archiving.git
   ```
2. Navigate to the project directory:
   ```bash
   cd BiblioTech-Algorithmic-Archiving
   ```

### Running the Application

You can use the provided PowerShell script to run the application:

```powershell
.\run_app.ps1
```

Or manually compiling and running the `src/application/Main.java`.

## Project Structure

- `src/`: Source code (Controllers, Models, Views, Services)
- `data/`: CSV data storage and configuration
- `lib/`: External libraries (JavaFX, etc.)
